-----------------------------------
 Paso a pao para correr el proyecto 
-----------------------------------

1. Requisitos

* XAMPP (Apache + MySQL) instalado y corriendo.
* PHP >= 8.2 (la versión en XAMPP debe ser compatible).
* Composer instalado.

2. Dónde poner el proyecto

* Copiar la carpeta del proyecto dentro de C:\xampp\htdocs\ClassProduct

3. Instalación de dependencias

* Abrir terminal en C:\xampp\htdocs\ClassProduct
* Ejecutar:
  composer install

4. Configuración inicial de la app

* Si no existe, crear (.env) desde (.env.example) con el comando:
  cp .env.example .env 

* Generar key:
  php artisan key:generate

5. Base de datos (MySQL con XAMPP)

* En phpMyAdmin (http://localhost/phpmyadmin) crear la base de datos:
  nombre: classproduct  (o el que prefiera)

* Verificar/editar .env:
  DB_CONNECTION=mysql
  DB_HOST=127.0.0.1
  DB_DATABASE=classproduct
  DB_USERNAME=root
  DB_PASSWORD=
  Ajustar el DB_CONNECTION, verifica que si sea mysql y ademas el nombre de la base de datos

6. Migraciones

* Ejecutar migraciones:
  php artisan migrate

* La migración se genera en la siguiente dirección:
  database/migrations/2026_02_22_203934_create_products_table.php

7. Cómo ejecutar la aplicación

* En la terminal:
  php artisan serve
* Acceder en:
  http://127.0.0.1:8000

8. Rutas importantes

* Página inicial:
  /routes
  (ver routes/web.php)

* Crear Producto:
  /products/create
  (vista en resources/views/products/create.blade.php)

* Listar Productos:
  /products/show
  (vista en resources/views/orders/index.blade.php)

* Modelo usado:
  App/Models/Product
  (app/Models/Product.php)

* Controlador:
  app/Http/Controllers/ProductController.php

9. Población

* Llenar todos los campos del formulario, los mas importante para tener en cuenta es que el campo 
  de imagen pide una URL de alguna imagen del producto que queremos crear.
